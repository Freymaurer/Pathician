PathfinderAttackSimulator

FullRoundAttackAction

Namespace: PathfinderAttackSimulator

Functions and values

Function or valueDescription
myFullAttack (...)
Signature: char:CharacterStats -> size:SizeType -> weapons:(Weapon * WeaponType) [] -> modifications:AttackModification [] -> unit []

This function returns the output of a full round attack action based on the used character stats, weapons and modifications. Weapons need an additional WeaponType: PrimaryMain for the weapon which should be used with things like haste, Primary for Primary natural attacks or two weapon fighting, and Secondary for secondary natural attacks.
